In Java , Recursion means a function calling itself to solve a complex problem.
A recursive function keeps calling itself until a stopping condition (base case) is reached.
Recursion is a technique where a method calls itself repeatedly to solve smaller parts of the same problem . Every recursive program has two parts - (I)Base Condition case - condition to stop recursion. (II) Recursive case - function calls itself
